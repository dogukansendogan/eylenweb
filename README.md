# Eyleniyoruz Villam'da

Eyleniyoruz Villam'da, villa kiralama işlemleri için geliştirilmiş modern bir web sitesidir. Kullanıcılar villaları inceleyebilir, konuma göre keşif yapabilir, öne çıkan villaları görüntüleyebilir ve platform üzerinden iletişim bilgilerine ulaşabilir.

Bu proje; kullanıcı dostu arayüz, modern tasarım, responsive yapı ve Firebase entegrasyonu hedeflenerek geliştirilmiştir.

---

## Kullanılan Teknolojiler

- Next.js
- React
- TypeScript
- Tailwind CSS
- Firebase
- Firestore
- Firebase Authentication
- Jest
- React Testing Library
- Playwright

---

## Özellikler

- Modern ve responsive ana sayfa
- Villa listeleme alanları
- Öne çıkan villalar bölümü
- Konum / lokasyon tanıtımları
- Kullanıcı dostu navigasyon yapısı
- Firebase bağlantısı
- Authentication altyapısı
- Test altyapısı
- E2E test desteği
- SEO uyumlu Next.js yapısı

---

## Kurulum

Projeyi bilgisayarınıza klonlayın:

git clone https://github.com/dogukansendogan/eylenweb.git

ardından terminalinizde;

cd eylenweb

npm install

npm run dev

http://localhost:3000 adresini açın
